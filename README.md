## Features
**1.** Bootstrap Components- Jumbotron Showcase, Scroll Down Effect and Font Awesome Icons.

**2.** LESS Preprocessor - Variables, Mixins, Operations.

**3.** WinLESS as the compiler. (Convert *.less* to *.css* & minify the file)
